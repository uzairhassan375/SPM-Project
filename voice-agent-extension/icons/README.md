# Icons Directory

This directory should contain the following icon files for the Chrome Extension:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create icons using:
- Online icon generators
- Image editing software (GIMP, Photoshop, etc.)
- Icon design tools

The icons should represent a microphone or voice-related theme to match the extension's purpose.

## Temporary Solution

If icons are not available, Chrome will use default extension icons. The extension will still function correctly without custom icons.
